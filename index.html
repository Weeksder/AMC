PPTX Notes Viewer

Open a PowerPoint file to view slides and speaker notes in a new tab. Works in the browser — no install required.

📑
Drop a .pptx here or click to choose a file

Tips
  • Each file opens in a new browser tab.
  • Allow pop-ups for this site if the tab is blocked.
  • Best with image-based slides (typical AMC decks) and speaker notes.
  • Notes edits stay in that browser unless you export later.
